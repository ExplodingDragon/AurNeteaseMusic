# ArchLinux-AUR  网易云音乐 

## 特性

- 可在线播放 FLAC 
- 最小化菜单显示修复
- UI 放大 1.25 倍

## 安装

```bash
git clone https://github.com/ExplodingDragon/AurNeteaseMusic.git
cd AurNeteaseMusic
makepkg -si
```
